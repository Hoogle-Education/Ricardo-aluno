# Estruturas de repetição

Nossa intenção, primeiramente, será: Fazer os programas repetirem processos em cima do código.

Elementos essenciais para estruturas de repetição:

```
1. inicialização
2. verificação
3. atualização
```

## Estruture `while` - *enquanto*

```
Enquanto( Condição ){
  Faça isso aqui
}
```

em java:
```java
while( /*verificação*/ ){
  // seu código
}
```
